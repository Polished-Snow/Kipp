description: Index and search reference repos for Kipp using Nia
---
When invoked as `/nia index <owner/repo>`, call Nia's `index_repository`
tool with that repo, then confirm status with `check_repository_status`.

When invoked as `/nia search <query>`, call Nia's `search_codebase` /
`search_documentation` tools scoped to the reference set below, and always
name which repo/file backs the answer.

Reference set to keep indexed for Kipp:
- antirez/ds4
- ggml-org/llama.cpp
- ggml-org/ggml
- ggml-org/whisper.cpp
- karpathy/llm.c
- GeeeekExplorer/nano-vllm
- vllm-project/vllm
- sgl-project/sglang
